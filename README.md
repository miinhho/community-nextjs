## Next.js + Community


### Tech Stack

1. **Framework**: Next.js  
2. **Database**: PostgreSQL  
3. **ORM**: Prisma  
4. **Auth**: Auth.js  
5. **Components**: Shadcn/ui  
6. **Styling**: TailwindCSS  
7. **Validation**: Zod  
8. **Text Editor**: Lexical  


### Features

**Auth**:  
  - 자체 로그인 / 로그아웃 / 회원가입

**OAuth**:  
  - Google
  - Naver
  - Kakao

**유저**:  
  - 프로필
  - 팔로우
  - 다이렉트 메세지
  - 알림 (친구 추가 요청, 메세지, 댓글)

**커뮤니티**:  
  - 글 작성 (이미지 첨부, 폰트 선택, 유튜브 동영상 첨부)
  - 글 좋아요
  - 댓글 작성
  - 댓글 좋아요
